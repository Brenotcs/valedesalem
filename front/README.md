# Vale de Salem - React App

Aplicação React para o site Vale de Salem - Artes Místicas em Miçanga.

## Instalação

```bash
npm install
```

## Executar em Desenvolvimento

```bash
npm start
```

A aplicação será aberta em [http://localhost:3000](http://localhost:3000).

## Build para Produção

```bash
npm run build
```

Os arquivos otimizados estarão na pasta `build/`.

## Estrutura do Projeto

```
front/
├── public/

├── src/
```

## Componentes

- **Header**: Cabeçalho com logo e tagline
- **Hero**: Seção principal com efeito de lua e estrelas
- **Features**: Grade de características do produto
- **Footer**: Rodapé com copyright
